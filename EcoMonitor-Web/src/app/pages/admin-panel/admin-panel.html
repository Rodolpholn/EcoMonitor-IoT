Configurações de IoT e Planta

Siga o passo a passo abaixo para configurar um novo sensor ESP32 no sistema EcoMonitor.

1
Configure o Firmware da ESP32

No código da ESP32, defina a URL da API: https://ecomonitor-iot-production.up.railway.app/api/Sensores e insira a API Key no header X-Api-Key. Certifique-se de que o Wi-Fi está configurado corretamente.

2
Faça Upload da Planta Baixa

Na tela de Sensores IoT, clique com o botão direito sobre o mapa e selecione "Upload da Planta". Insira a URL pública da imagem da planta baixa (PNG, JPG ou SVG). A imagem será exibida como fundo do mapa.

3
Posicione o Sensor na Planta

Ainda na tela de Sensores IoT, clique com o botão direito no local exato da planta onde o sensor está instalado fisicamente. Selecione "Adicionar Equipamento", informe o ID da ESP32 (ex: ESP32-01) e o nome do local (ex: Entrada). O sensor será fixado nessa posição.

4
Ligue a ESP32

Com o sensor pré-cadastrado no sistema, ligue a ESP32. Ela irá se conectar ao Wi-Fi e começar a enviar leituras automaticamente. O ícone do sensor na planta ficará ativo e exibirá os dados em tempo real ao passar o mouse.

5
Verifique no Dashboard

Acesse o Dashboard para confirmar que os dados estão chegando. O indicador SISTEMA ONLINE aparecerá quando a última leitura tiver menos de 60 segundos.
