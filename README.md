# 🌱 Ecomonitor IoT

Sistema de monitoramento inteligente para coleta, processamento e visualização de dados em tempo real através de dispositivos IoT.

---

## 📌 Sobre o projeto

O **Ecomonitor IoT** foi desenvolvido para resolver a necessidade de monitoramento contínuo de dados em ambientes que exigem tomada de decisão rápida e baseada em métricas.

A solução permite integrar dispositivos IoT a uma plataforma web, exibindo informações em tempo real através de dashboards interativos.

---

## 🚀 Funcionalidades

✔ Coleta de dados em tempo real via dispositivos IoT  
✔ API REST para comunicação entre backend e frontend  
✔ Dashboard com gráficos e métricas dinâmicas  
✔ Armazenamento estruturado de dados  
✔ Visualização de histórico de informações  

---

## 🏗️ Arquitetura

O sistema segue uma arquitetura baseada em APIs, permitindo escalabilidade e fácil integração com outros sistemas.

---

## 🛠️ Tecnologias utilizadas

- **Frontend:** Angular  
- **Backend:** C# / .NET  
- **Comunicação:** APIs REST  
- **Banco de dados:** SQL  
- **Integração:** Dispositivos IoT  

---

## 📊 Caso de uso

Aplicável em cenários como:
- Monitoramento ambiental  
- Controle industrial  
- Gestão de sensores remotos  
- Análise de dados em tempo real
